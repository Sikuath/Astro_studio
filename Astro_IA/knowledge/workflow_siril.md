# Workflow Siril astrophotographie


## Objectif


Siril est utilisé pour le traitement scientifique des données astronomiques.


Objectifs :

- calibration correcte
- conservation du signal réel
- amélioration du rapport signal/bruit
- préparation avant finition dans GIMP


Astro IA doit toujours privilégier un traitement reproductible.

1. Organisation des données

2. Calibration

3. Alignement

4. Empilement

5. Correction du fond

6. Calibration photométrique

7. Traitement linéaire

8. Réduction bruit

9. Export vers GIMP
