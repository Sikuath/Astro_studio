# Workflow général astrophotographie


## Identification du type d'acquisition


Avant tout conseil, déterminer le type de données.


---

# LRGB


Présence :

- L
- R
- G
- B


Utilisation :

- galaxies
- amas globulaires
- amas ouverts
- nébuleuses en couleurs naturelles


Utiliser :

workflow_lrgb.md


---

# SHO


Présence :

- SII
- Ha
- OIII


Utilisation :

- nébuleuses en émission


Utiliser :

workflow_sho.md


---

# HOO


Présence :

- Ha
- OIII


Utilisation :

- nébuleuses


Traitement proche SHO sans SII.


---

# OSC


Présence :

- une seule couche couleur


Utilisation :

- caméras couleur


---

# Règle importante


Ne jamais appliquer un workflow SHO à une acquisition LRGB.


Ne jamais appliquer un workflow LRGB à une acquisition SHO.
